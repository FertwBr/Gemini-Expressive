# Gemini Expressive

A browser extension to improve the Gemini web interface with a navigation timeline, collapsible code blocks, and dynamic Material You theming.

## Features

* **Timeline Navigation:** Adds a floating timeline on the right side to easily navigate through long conversations.
* **Code Block Collapser:** Allows you to collapse and expand long code blocks to save screen space.
* **Enhanced Code Headers:** Displays programming language icons and extracts file names for code blocks.
* **Material 3 Expressive Theming:** Customize your experience with Light, Dark, or System default modes, and pick a dynamic seed color to generate a custom Material You palette. Includes predefined expressive color swatches.
* **Multi-language Support:** Available in English, Portuguese, Spanish, German, Hindi, and Japanese.
